# GDGoC WebDev - Boy Steven Benaya Aritonang

This repository contains my submission for **GDGoC Web Development Task 1**.

## Project

**Boy's Lab** is a simple event registration mini site built with:

- HTML
- CSS
- JavaScript

## Features

- Clean and responsive layout
- Event information section
- Registration form
- Custom experience-level dropdown
- Live registered participants note
- Local storage support for saved participants

## Folder Structure

```text
tugas-1/
├── index.html
├── style.css
└── script.js
```

## How to Run

Open `tugas-1/index.html` directly in a browser.

Or run a local server:

```powershell
cd tugas-1
python -m http.server 8000
```

Then open `http://localhost:8000`.
